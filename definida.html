Integración Definida

¿Qué es una integral definida?

La integral definida mide la acumulación entre dos puntos: \(\int_a^b f(x)\,dx\). Por el Teorema Fundamental del Cálculo, si \(F\) es antiderivada de \(f\),

\(\int_a^b f(x)\,dx = F(b) - F(a).\)

Geométricamente se interpreta como el área algebraica entre la curva y el eje \(x\) desde \(x=a\) hasta \(x=b\). Si la función está debajo del eje, el resultado será negativo.

Propiedades clave
  • Linealidad: \(\int_a^b (c f + g) = c\int_a^b f + \int_a^b g\).
  • Suma en intervalos: \(\int_a^b + \int_b^c = \int_a^c\).
  • Cambio de orientación: \(\int_a^b = -\int_b^a\).
  • Simetría: si \(f\) es par, \(\int_{-a}^a f = 2\int_0^a f\); si es impar, \(\int_{-a}^a f = 0\).
Ejemplo rápido \(\int_0^2 (3x^2 - 1)\,dx = \Big[x^3 - x\Big]_0^2 = (8-2)-0 = 6\)

Interpretación visual

Puedes imaginar la integral definida como pintar el área bajo la curva entre dos límites. Si la curva cruza el eje, el área de arriba se suma y la de abajo se resta.

Ejemplo gráfico La integral de \(f(x)=x\) entre 0 y 2 es un triángulo de base 2 y altura 2: Área = \(\tfrac{1}{2}\times 2\times 2 = 2\).

Ejemplos paso a paso

Ejemplo 1 (básico) \(\int_0^3 (2x+1)\,dx\) Antiderivada: \(F(x)=x^2+x\). Evaluamos: \(F(3)-F(0) = (9+3)-(0)=12.\)
Ejemplo 2 (sustitución) \(\int_0^1 2x e^{x^2} dx\) Sea \(u=x^2 \Rightarrow du=2x dx\). Límites: \(u(0)=0,\; u(1)=1\). \(\int_0^1 e^u du = e^1 - e^0 = e-1\).
Ejemplo 3 (área entre curvas) \(\int_0^1 (x - x^2)\,dx\) Antiderivada: \(\tfrac{x^2}{2} - \tfrac{x^3}{3}\). Evaluamos: \((\tfrac{1}{2}-\tfrac{1}{3})= \tfrac{1}{6}.\)

Aplicaciones

  • Área bajo la curva: \(A = \int_a^b f(x)\,dx\).
  • Área entre curvas: \(A = \int_a^b (f(x)-g(x))\,dx\).
  • Volúmenes de revolución: \(V=\pi\int_a^b (R(x))^2 dx\).
  • Promedio de una función: \(f_{prom}=\tfrac{1}{b-a}\int_a^b f(x)\,dx\).
  • Trabajo: \(W=\int_a^b F(x)\,dx\).
Ejemplo de promedio \(\tfrac{1}{\pi}\int_0^\pi \sin x \,dx\) Antiderivada: \(-\cos x\) Evaluar: \((-(-1))-(-1))=2\) Promedio: \(\frac{2}{\pi}\)

Casos especiales

Algunas integrales se simplifican por la simetría de la función:

Ejemplo (función impar) \(\int_{-3}^3 x^3 dx = 0\) porque el área de la izquierda y la derecha se cancelan.
Ejemplo (función par) \(\int_{-2}^2 x^2 dx = 2\int_0^2 x^2 dx\). Antiderivada: \(\tfrac{x^3}{3}\). Evaluar: \(2\cdot (\tfrac{8}{3}-0)=\tfrac{16}{3}\).

Tips y errores comunes

  • No olvides cambiar los límites en un cambio de variable.
  • El área bajo el eje se cuenta como negativa.
  • Las propiedades de simetría ahorran tiempo.
  • Una integral definida no siempre representa un área: también puede dar trabajo, volumen o promedio.
